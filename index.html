gogsi

A game state integration library for Dota 2, written in Go.

View on GitHub Download .zip Download .tar.gz

The Dota 2 client is capable of streaming json updates about the game to an arbitrary HTTP endpoint, intended originally for applications like the TI5 stage and LED-enabled peripherals.

gogsi listens to these updates and applies a user-defined callback to each one.

Barebones Example

package main

import "github.com/mammothbane/gogsi"

func main() {
    gogsi.Listen("localhost:3000", func(s *gogsi.State) error {
        if s.Added.Map.NightstalkerNight {
            /* adjust your phillips hue light, play a scary noise, update statistics, log something, etc. */
        }
        return nil
    })
}
